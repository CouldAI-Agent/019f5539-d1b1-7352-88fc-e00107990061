# Pañol Escolar

Una aplicación profesional para la administración de un pañol escolar, donde los profesores solicitan herramientas para sus clases y posteriormente registran la devolución de las mismas. 

## Características principales
- **Dashboard en tiempo real**: Visualiza el total de herramientas, préstamos activos y alertas de bajo stock.
- **Solicitud de préstamos**: Registra fácilmente qué herramienta se lleva cada curso, con cantidad y observaciones.
- **Devoluciones ágiles**: Gestiona las herramientas devueltas, actualizando el inventario al instante.
- **Diseño Moderno**: Interfaz inspirada en Windows 11 usando Material Design 3, optimizada para uso rápido y sin complicaciones.
- **Multiplataforma**: Funciona de forma fluida en Web, Escritorio y Dispositivos Móviles gracias a un diseño verdaderamente responsivo.

## Stack Tecnológico
- **Frontend**: Flutter (Dart)
- **Diseño**: Material Design 3

## Instalación y ejecución
1. Asegúrate de tener instalado Flutter en tu sistema.
2. Clona este repositorio o descarga el código fuente.
3. En la raíz del proyecto, instala las dependencias:
   ```bash
   flutter pub get
   ```
4. Ejecuta la aplicación en la plataforma deseada:
   ```bash
   flutter run
   ```
